# 버튼 잡기 게임 / C#

대학 과제로 작성한 C# 프로젝트입니다.

제한시간 내에 버튼을 최대한 여러번 클릭해(잡아) 높은 점수를 얻는 게임입니다.

해로운 버튼을 클릭해 점수를 획득하세요!

버튼을 클릭할 때마다 버튼이 무작위 위치로 이동합니다.

순발력 있게 마우스를 이동하는 게 좋겠죠.
